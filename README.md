# Anti-DDoS Tool

## Installation
1. Clone the repository:
   
   ```bash
   git clone https://github.com/nrokami/anti-ddos-nro.git
   cd anti-ddos-nro
   ```

2. Install the dependencies:
   
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the anti-DDoS tool, execute the following command:

```bash
python anti_ddos.py
```

## Configuration
You may need to configure the settings by editing the `config.json` file to set your desired thresholds and options.
